Technologies Used

  1. Angular 5 / angular-cli

Testing Framework

  1. Karma - installed karma-spec-reporter for more detailed test results in command.

  2. Protractor - End to End testing

  3. Jasmine

Custom Error Handling

Custom XyzErrorHandlerService to overwrite the default ErrorHandler. This service prints out timestamp, error message, error zone, and errort task to the console for a more detailed message.
